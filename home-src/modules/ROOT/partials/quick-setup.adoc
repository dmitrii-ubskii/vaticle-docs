== Create a new database

To create a new database:

. Click the image:{page-component-version}@home::studio-icons/svg/studio_dbs.svg[width=24] btn:[Manage Databases] button in the top toolbar.
. Enter a name for a new database and click btn:[Create].
. Select the new database from the btn:[Select Database] dropdown (image:{page-component-version}@home::studio-icons/database-none.png[]) in the top toolbar.
