= TypeDB Community Edition

Connecting to TypeDB CE requires:

* `address` and `port` of the server (defaults: `127.0.0.1` and `1729`)
* a valid `username` and `password` (default credentials: `admin` and `password`)
* optionally, a valid TLS certificate for in-flight encryption (not used here)
