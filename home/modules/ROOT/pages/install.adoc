[#_server]
== Server

TypeDB runs in the following environments:

- https://cloud.typedb.com[TypeDB Cloud]: The fully managed service for TypeDB deployments in the cloud
- xref:manual:ROOT:self-managed/cloud.adoc[TypeDB Cloud Self-hosted]: The subscription-based, self-managed version of TypeDB
- xref:manual:ROOT:self-managed/index.adoc[TypeDB Open Source]: The open source, free-to-use, and self-managed version of TypeDB


[#_studio]
== TypeDB Studio

Install a complete IDE that facilitates development on TypeDB with a schema manager, result visualizer, and more.

[tabs]
====
macOS::
+
--
Install TypeDB Studio using Homebrew:

[source,console]
----
brew tap typedb/tap
brew install --cask typedb/tap/typedb-studio
----

Or install manually:

. Download the right package for your system:
include::manual:resources:partial$typedb-studio-latest-links.adoc[tag=mac]
+
. Open the downloaded `.dmg` file and accept the license agreement.
. Drag and drop TypeDB Studio executable to the Applications folder.

Older versions of TypeDB Studio are available from the https://cloudsmith.io/~typedb/repos/public-release/packages/?q=name%3A%5Etypedb-studio-mac&sort=-version[Downloads repository].
--

Linux::
+
--
Install TypeDB Studio using `apt`:

. Add the TypeDB repository:
+
[source,console]
----
sudo apt install software-properties-common apt-transport-https gpg
gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-key 17507562824cfdcc
gpg --export 17507562824cfdcc | sudo tee /etc/apt/trusted.gpg.d/typedb.gpg > /dev/null
echo "deb https://repo.typedb.com/public/public-release/deb/ubuntu trusty main" | sudo tee /etc/apt/sources.list.d/typedb.list > /dev/null
sudo apt update
----
. Install the application:
+
[source,console]
----
sudo apt install typedb-studio
----

Or install manually:

. Download the right package for your system:
include::manual:resources:partial$typedb-studio-latest-links.adoc[tag=linux]
+

. Extract the archive with TypeDB Studio into a new directory, for example:
+
[source,console]
----
mkdir /opt/typedb-studio
tar -xzf ~/Downloads/<filename>.tar.gz -C /opt/typedb-studio
----
+
Where `<filename>` is the name of the archive.
. Add a symlink to the TypeDB Studio executable in the `/usr/local/bin` directory, for example:
+
[source,console]
----
ln -s /opt/typedb-studio/<filename>/typedb-studio /usr/local/bin/typedb-studio
----

Older versions of TypeDB Studio are available from the https://cloudsmith.io/~typedb/repos/public-release/packages/?q=name%3A%5Etypedb-studio-linux&sort=-version[Downloads repository].
--

Windows::
+
--
Download and install TypeDB Studio:

. Get the installer:
include::manual:resources:partial$typedb-studio-latest-links.adoc[tag=windows]
+
. The installation wizard will guide you through the process.

Older versions of TypeDB Studio are available from the https://cloudsmith.io/~typedb/repos/public-release/packages/?q=name%3A%5Etypedb-studio-windows&sort=-version[Downloads repository].
--
