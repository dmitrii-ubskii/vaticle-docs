== Update data

To update data in a database, we match the existing data, delete it, and then insert new data to replace it. The following Update query updates the status of the order with ID "o0006". Using a `data` session and `write` transaction, image::{page-component-version}@home::studio-icons/svg/studio_run.svg[width=24] btn:[run] this query, then image::{page-component-version}@home::studio-icons/svg/studio_check.svg[width=24] btn:[commit] the transaction.

[,typeql]
----
match
  $order-6 isa order,
    has id "o0006",
    has status $old-status;
delete
  $order-6 has $old-status;
insert
  $order-6 has status "dispatched";
----

After committing the transaction, the updated information is persisted in the database and becomes available for querying. If we then re-run the above Fetch query, we should see the results have updated to reflect the change.
